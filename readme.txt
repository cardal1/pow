首先把程序源文件下载下来
然后进入到程序目录
进行npm install操作，把相关的包下载下来。
在进行node ./koa.js操作，把服务接口打开。
然后可以直接在浏览器上对3000端口上接口访问，主要接口有以下几个。

/create 创建一条主链，注意如果已经创建过主链的话就不可以继续创建，除非把当前链删除。
/showAll 查看当前链上的交易信息。
/add/:data 往链上申请增加一笔交易，data表示交易信息，一般能在5s内得到响应。（这个操作要在create创建链之后进行，否则会有相应提示创建一条链）
/clean 删除当前生成的链。

note:在运行后，本地目录会生成一个blocks.txt文件，里面记录着链上的区块信息，可以直接打开查看。程序默认的挖矿难度是5，如果想要更改挖矿难度，可以手动更改main.js文件下的difficulty的值。
